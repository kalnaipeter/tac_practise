# AI Documentation

Place 3rd party documentation, API references, and quickstart guides here so agents have accurate reference material instead of relying on (potentially outdated) training data.

## What Goes Here

- SDK quickstart guides for libraries used in the project
- API reference snippets for external services
- Framework documentation excerpts relevant to the codebase
- Any reference material an agent would need to avoid hallucinating API usage

## Why

Your agent is brilliant, but blind. It starts each session with no memory. Providing accurate, current documentation ensures it uses APIs correctly — especially for recently changed or niche libraries.
