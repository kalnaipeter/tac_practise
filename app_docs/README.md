# App Documentation

Feature documentation generated by the ADW document phase (`adw_document.py`). Each doc captures what was built, how it works, and how to use it — based on git diff analysis and the original specification.

## Assets

Screenshots and images are stored in `assets/`.

## Conditional Docs

After each feature is documented, an entry is added to `.github/prompts/conditional-docs.prompt.md` so future agents can find relevant documentation when working on related tasks.
